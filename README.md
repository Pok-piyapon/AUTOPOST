
# AUTO POST
โปรเจ็คนี้ทำขึ้นมาเพื่อเป็นผลงานและสามารถนำไปต่อยอดได้ตามต้องการ


## Installation

Install autopost with npm

```bash
  npm install
```

## Run Program

Install autopost with npm

```bash
  npm start
```
